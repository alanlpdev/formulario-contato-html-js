Formulário de Contato (HTML, CSS e JavaScript)

Este projeto é um formulário de contato moderno e funcional , desenvolvido com HTML, CSS e JavaScript. Ele permite que os visitantes enviem mensagens diretamente para o responsável pelo site, de forma rápida, segura e organizada.

Funcionalidades

Campos obrigatórios: Nome , E-mail , Assunto e Mensagem

Validação em tempo real com JavaScript (evita envio de campos vazios ou e-mails inválidos)

Envio de dados via EmailJS , sem necessidade de backend

Feedback visual ao usuário com mensagens de sucesso ou erro

Objetivo

Oferecer uma forma prática e profissional de contato direto com o administrador do site, ideal para portfólios, landing pages, sites empresariais ou pessoais.

Tecnologias Utilizadas

HTML5

CSS3 (Flexbox e estilos modernos)

JavaScript (DOM, validação, integração com EmailJS)

EmailJS (serviço gratuito para envio de e-mails via frontend)
